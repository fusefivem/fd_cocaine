# FD Cocaine
<img width="2700" height="1500" alt="fd_cocaine" src="https://github.com/user-attachments/assets/10aacd96-9b2e-423b-bdbe-59c6d73c56ec" />

A high performance cocaine harvesting and production system for ESX, built for ox_lib and ox_inventory.

## Features
- **Harvesting:** Players can gather coca leaves at locations easily defined via the configuration file.
- **Immersive Processing:** Features an advanced production system utilizing synchronized scenes for realistic roleplay. 
- **OX Based:** Fully optimized for ox_lib  and ox_inventory.

## Requirements
- ESX Framework (`es_extended`)
- `ox_lib`
- `ox_inventory`
- `ox_target`

## Documentation
- For detailed setup and configuration, visit: [docs.fusedev.eu/cocaine](https://docs.fusedev.eu/cocaine)

## Installation
1. Download the script
2. Put the `fd_cocaine` folder into your `resources` directory
4. Open `config.lua` and configure settings to your preference
5. Add `ensure fd_cocaine` to your `server.cfg`
6. Add **ALL** items to ox_inventory/data/items.lua
7. Restart your server
